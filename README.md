# Generational Seed Corporation Website

Modern institutional website for Generational Seed Corporation - a governance-first enterprise ecosystem.

## Features
- 🎨 Modern UI with Tailwind CSS
- ✨ Smooth animations with Framer Motion
- 📱 Fully responsive design
- 🔒 hCaptcha integration
- 📧 Contact form with email notifications
- 🎬 Hero carousel with auto-rotation
- 📊 Scroll-based animations

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- hCaptcha
- Backend: Node.js/Express (separate repo)

## Setup
```bash
npm install
npm run dev
```

## Environment Variables
```
VITE_API_URL=your_backend_url
VITE_HCAPTCHA_SITE_KEY=your_hcaptcha_key
```

## Deployment
Deploy to Vercel at gs-corp.vercel.app. Once deployed, add the domain to backend CORS in Render.
